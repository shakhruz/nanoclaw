# doc-invoice — Счёт на оплату / Акт выполненных работ

Создаёт счёт на оплату или акт в HTML (print-ready) и публикует на `milagpt.cc/c/<client>/invoice-<title>-<secret>`.

## Когда использовать

- «Выставь счёт для X на $500»
- «Сделай акт выполненных работ для Y»
- «Нужен счёт-фактура»

## Правила оформления

- Язык: **русский**, обращение **«вы»**
- Шрифт: Georgia / Times New Roman (финансовый документ)
- **Эталон:** `/workspace/group/clients/vladlen-ai-personal/` (счёт в ledger)
- Суммы: **USD** + «курс ЦБ РУз на дату оплаты»
- Прописью: сумма цифрами и словами (напр. «Пятьсот долларов США 00 центов»)
- Срок оплаты: явно указать дату (напр. «до 30 апреля 2026 г.»)
- Печать/подпись: место для физической подписи + круг-заглушка под печать

### Print / PDF-ready (обязательно)

Счёт отправляется в бухгалтерию и должен аккуратно печататься на A4.

## Типовые формулировки назначения платежа

- Первый взнос (50%): «Предоплата 50% по договору № __ от __.__. 2026 г.»
- Остаток (50%): «Оплата оставшейся части (50%) по договору № __ от __.__. 2026 г.»
- Ежемесячная: «Абонентская плата за [месяц] 2026 г. по договору № __ от __.__. 2026 г.»

---

## Что нужно получить

- Покупатель: имя/название, адрес, тел, контактное лицо
- ИНН заказчика (для юрлица)
- Услуга: описание, кол-во, цена, сумма
- Общая сумма договора + уже оплачено + к оплате
- Срок оплаты

## Алгоритм

1. Определи номер счёта (посмотри ledger — какой был последний)
2. Напиши HTML в `/workspace/group/clients/<client-slug>/invoice-<N>.html`
3. Опубликуй:
   ```bash
   /home/node/.claude/skills/web/publish-client-doc-api.sh \
     <client-slug> invoice <title> <html-file> "<заметка>"
   ```
4. Отправь ссылку Ашоту

```
  Поставщик: ИП Шарапова...     | Покупатель: [Имя/Компания]
  реквизиты...                   | контакты...

Основание: ссылка на договор

Таблица услуг: №, Наименование, Кол-во, Цена USD, Сумма USD

Расчёт к оплате:
  Общая стоимость: $X,XXX
  Предоплата (оплачено): −$X,XXX   ← если частичная оплата
  Итого к оплате: $XXX (жирный)

Сумма прописью + курс ЦБ РУз

Желтый блок: Срок оплаты + статус услуги

Реквизиты для оплаты (highlight-box):
  Получатель, ИНН, Р/с, Банк, Назначение платежа

Подписи: 2 колонки с линиями

Footer: Документ сформирован [дата] · AshotAI / MILA GPT · ashotai.uz
```

## Акт выполненных работ

Структура аналогична счёту, но:
- Заголовок: АКТ ВЫПОЛНЕННЫХ РАБОТ № N

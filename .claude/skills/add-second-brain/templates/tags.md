# Typed Tag System

Whitelist тегов для классификации информации. Используй inline на любой странице вики и в записях `log.md`.

Lint-проверка ругается если встречает тег вне этого whitelist'а — добавь сюда новый тег с описанием прежде чем использовать.

---

## Tag Definitions

| Тег | Использование | Пример |
|---|---|---|
| `#DECISION` | Явные решения, принятые тобой или командой | `#DECISION 2026-04-21 — Внедряем второй мозг через /add-second-brain` |
| `#PREFERENCE` | Личные предпочтения, стилистические выборы | `#PREFERENCE: отвечать кратко, без лишней воды` |
| `#MILESTONE` | Ключевые достижения, вехи | `#MILESTONE: Wiki v1.0 запущена 2026-04-21` |
| `#PROBLEM` | Известные проблемы, блокеры, нерешённые вопросы | `#PROBLEM: telegram-ads cookie expired check ломается` |
| `#CONTACT` | Ключевые контактные данные людей | `#CONTACT: @username, роль, контекст знакомства` |
| `#ACTION` | Запланированные действия, следующие шаги | `#ACTION: запустить A/B-тест объявлений до 25 апреля` |
| `#INSIGHT` | Нетривиальные выводы, озарения | `#INSIGHT: 95% экспертов не упакованы — это и есть рынок` |
| `#WARNING` | Риски, юридические нюансы | `#WARNING: нельзя использовать медицинские термины в рекламе` |

---

## Использование в страницах

Добавляй теги inline в текст:

```markdown
## Решения

> #DECISION 2026-04-21 — Используем essentials.md вместо полного index.md при старте сессий

> #MILESTONE 2026-04-21 — Second Brain установлен через /add-second-brain v1.0.0

> #PROBLEM CTR русскоязычной рекламы 0% — нужен узбекский текст
```

## Использование в `log.md`

В заголовках записей:

```markdown
## [2026-04-21] milestone | Wiki v1.0 запущена #MILESTONE #DECISION
```

Lint парсит inline-теги и строит граф тегов — это помогает находить кросс-категориальные паттерны.

---

## Расширение whitelist'а

Добавь новый тег только когда:
1. Существующие 8 не покрывают семантику
2. Тег используется минимум 3 раза в реальных pages
3. Описание + пример понятны без контекста

Не плоди тегов на каждый чих — иначе теряется навигационная ценность.
